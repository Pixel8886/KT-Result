# KT-Result

## Описание
Данное задание выполнено с использованием XSLT версии 1.0 и XPath версии 1.0.

Исходный XML-документ (`source.xml`) содержит вложенную структуру XHTML
в виде списка комментариев.  
В результате преобразования формируется плоский XML-документ
с элементами `<item>` и сохранением иерархии через атрибут `parentid`.

## Файлы
- `source.xml` — исходный XHTML-документ
- `transform.xsl` — XSLT-преобразование
